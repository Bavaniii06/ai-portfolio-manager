# 🚀 AI Portfolio Manager

**AI Portfolio Manager** is a financial intelligence platform designed for modern wealth management. It leverages real-time market data and AI-driven insights to provide sophisticated SIP planning and stock discovery.

## 💎 Features

- **🧠 AI Advisory Engine**: Real-time monitoring of 500+ NSE assets with dynamic risk-profile mapping.
- **⚡ SIP Planner**: Multi-asset discovery hub featuring **Stability**, **Growth**, **Alpha Stocks**, and **Commodities** (Gold/Silver).
- **🏙️ Professional Naming**: Automated name cleaning to strip technical noise (ETF, BEES, .NS) for a premium user experience.
- **🛡️ Risk-Anchored Suggestions**: Suggestions that update instantly based on your **Age**, **Investment Horizon**, and **Risk Appetite**.
- **📊 Real-Time Simulations**: High-fidelity backtesting for both individual stocks and mutual funds (ETFs).

## 🛠️ Technology Stack

- **Frontend**: Streamlit (Dark Mode UI)
- **Data Engine**: yfinance & Pandas Intelligence
- **Visualization**: Plotly Interactive Charts
- **Backend**: Python 3.10+ Reactive Logic

## 🚀 Getting Started

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Bavaniii06/ai-portfolio-manager.git
    cd ai-portfolio-manager
    ```
2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Launch the Dashboard**:
    ```bash
    streamlit run ui/dashboard.py
    ```
